# Notification Sound

## Required File

Please add a notification sound file named `notification.mp3` to this directory.

## Recommendations

The sound should be:
- **Short**: 0.5-1 second duration
- **Subtle**: Not too loud or jarring
- **Pleasant**: Gentle "ding" or "pop" sound

## Where to Find Free Notification Sounds

1. **Pixabay** - https://pixabay.com/sound-effects/search/notification/
   - Free, no attribution required
   - Good selection of notification sounds

2. **Freesound** - https://freesound.org/search/?q=notification
   - Creative Commons licensed
   - Wide variety

3. **Zapsplat** - https://www.zapsplat.com/sound-effect-category/notifications/
   - Free with attribution
   - Professional quality

## Alternative: Use a Data URI

If you want to generate a simple beep programmatically, you can update the hook to use a generated tone instead of an external file.
